An analysis of character data in Umamusume: Pretty Derby and historical data of their real life racehorse counterparts for a university data science course. The main motive of this project was to identify qualitative evidence of how Umamusume takes nuanced and extensive inspiration from Japanese horse racing (e.g. Umamusume character heights being based off of real life horse weights). Queries were done pertaining to the distribution of certain variables in the population of racehorses represented in Umamusume, as well as tests on possible correlations between variables in the game data and historial racehorse data. 

All code in sqlite_helpers.py is credited to course instructor Jennifer Campbell and used with permission.
## Data Files
**cleaned.db:** Simplified version of master.mdb containing only tables used in the project.

**master.mdb:** Database file extracted from version 2.24.1 of the Japanese release of Uma Musume: Pretty Derby. 

**netkeiba.csv:** CSV file of data used in the project scraped from [en.netkeiba.com](https://en.netkeiba.com).

**project_db.sqlite:** Database file generated and queried in the analysis process.
